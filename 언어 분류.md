# 마크업 언어
- 태그 등을 이용하여 문서나 데이터의 구조를 정의하는 언어
- 간단한 태그를 사용하여 문서요소 정의
- HTML(논란 중), XML, Markdown 등

# 프로그래밍 언어
- 주로 컴퓨터에게 명령을 내리고 작업을 수행하기 위해 사용
- 일련의 명령과 알고리즘을 구현하고, 프로그램을 만드는 데 사용
- 명령문, 함수, 변수 등을 사용하여 프로그램 구축
- Python, Java, C++, JavaScript 등
<br><br>
